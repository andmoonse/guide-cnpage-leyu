# guide-cnpage-leyu

本仓库用于归档和发布多个独立 HTML 页面。这些页面内容彼此独立，不针对任何单个域名或具体网站，仅作为静态页面集合进行版本管理与分发。

## 项目简介

该项目是一个轻量级的静态页面归档仓库。每个 HTML 文件均为完整独立的网页，可直接在浏览器中打开使用。仓库旨在提供一种简洁、透明的方式，保存和分享各类实用性或展示性的网页内容。

## 目录结构

```
guide-cnpage-leyu/
├── README.md          # 本说明文件
├── index.html         # 可选首页 / 索引页
└── pages/             # 存放各独立页面的目录
    ├── page1.html
    ├── page2.html
    └── ...
```

- `pages/` 目录下每个 HTML 文件均代表一个独立的页面，命名方式遵循页面主题或用途。
- 根目录下的 `index.html` 为可选项，用于展示页面列表或提供导航。

## 页面归档说明

- 所有页面均为纯前端 HTML，不依赖后端服务。
- 页面资源（如 CSS、JavaScript、图片等）均内嵌或使用相对路径引用。
- 归档内容可能不定期更新，新增页面或修正已有页面。
- 每个页面文件头部包含基本元信息（如标题、编码声明），便于识别。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问。
